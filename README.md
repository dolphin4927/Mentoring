### 🎃**참여 방법**
1. 해당 프로젝트 fork
2. fork된 자신의 원격 저장소 확인
3. 알고리즘 풀이 후 원격 저장소에 PUSH
4. Pull Request(PR) 요청
5. 스터디 참가자들 코드 리뷰 진행
6. 코드 리뷰 완료 후 MERGE
